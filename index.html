What Is Karumi-Env and How it can help me?

Karumi-Env is the web stack that we use at Karumi for our own servers and client solutions. We use Amazon Web Services (AWS) and in particular Elastic Beanstalk to setup.

The baseline is operating system Amazon Linux AMI 2013.09.2 running Python 2.7 on top of Apache, we added some feature on top and some modification.

Pre-installed Packages

  • Sass is a CSS extension language.
  • yUglify is a wrapper around UglifyJS and cssmin with the default YUI configurations on each of them.
  • libxml2 and libxslt as requirements for lxml.
  • libmemcached, zlib, cyrus-sasl-lib as requirements for pylibmc.

Apache Configuration Changes

  • Activate deflate(compressed output) config for non-binary files.
  • Activate explicit expire headers.
  • HTTP authorisation headers are passed through to WSGI.
